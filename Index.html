Preguntas sobre CSS


  • 1. ¿Qué es CSS?
    CSS (Cascading Style Sheets) es un lenguaje utilizado para describir la apariencia de un documento HTML.

  • 2. ¿Cuál es la función principal de CSS?
    Controlar el diseño visual de una página web, como colores, fuentes y la disposición de los elementos.

  • 3. ¿Cómo se puede incluir CSS en un documento HTML?
    En línea, en el encabezado del documento o en un archivo externo.

  • 4. ¿Qué son los selectores en CSS?
    Son los elementos que se utilizan para aplicar estilos a un grupo de elementos HTML.

  • 5. ¿Qué son las propiedades y valores en CSS?
    Las propiedades definen qué aspecto de un elemento cambiará, y los valores son las configuraciones específicas.

  • 6. ¿Cómo se aplica el modelo de caja en CSS?
    El modelo de caja describe los márgenes, bordes, rellenos y el contenido de un elemento.

  • 7. ¿Qué es el "cascading" en CSS?
    Es el mecanismo por el cual las reglas CSS se aplican en orden de especificidad y jerarquía.

  • 8. ¿Qué son las media queries?
    Son reglas que permiten aplicar estilos en función del tamaño de la pantalla o del dispositivo.

  • 9. ¿Cuáles son algunos estilos comunes que se pueden aplicar?
    Cambiar el color, la fuente, los márgenes, el padding, los bordes y la alineación.

  • 10. ¿Cómo se puede usar CSS para crear un diseño responsivo?
    Utilizando media queries y unidades relativas como porcentajes o "em", junto con técnicas como Flexbox o Grid.
